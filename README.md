## Executando

1. [Instale o Flutter](https://flutter.dev/docs/get-started/install)

2. Clone este repositório

```
git clone https://github.com/hb57786/todoey_flutter.git
```

3. Baixe as dependências

```
flutter pub get
```

4. Execute

```
flutter run
```
